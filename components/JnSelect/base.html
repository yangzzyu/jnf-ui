1、代码示例 ​

下拉选择组件————可实现单选多选（多选可使用全选功能）

html
<t-select
  placeholder="请选择工序"
  v-model="selectVlaue"
  :optionSource="state.stepList"
  valueKey="label"
  @change="selectChange"
/>
<t-select
  placeholder="请选择工序"
  v-model="selectVlaue"
  :optionSource="state.stepList"
  valueKey="label"
  @change="selectChange"
/>

2、配置参数（Attributes）继承 el-select Attributes ​

参数说明类型默认值
v-model绑定值boolean / string / number/Array无
multiple是否多选Booleanfalse
optionSource下拉数据源Array无
valueKey传入的 option 数组中，要作为最终选择项的键值 keyString'key'
labelKey传入的 option 数组中，要作为显示项的键值名称String'label'
